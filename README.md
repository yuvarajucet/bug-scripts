``` git clone https://github.com/yuvarajucet/spffinder.git```
## Step 2:

``` cd spffinder```

## Step 3:
``` chmod u+x spffinder```

## Step 4:
### For single domain
```python3 spffinder -s example.com```

## Step 5:
### For multiple domain contains txt file
```python3 spffinder -f filepath/newdir/file.txt```
